# Calculator

A simple web-based calculator built with **HTML**, **CSS**, and **JavaScript**.

Supports the following operations:
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Exponentiation (`^`)
